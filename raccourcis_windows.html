Action Raccourci clavier Windows
Copier (un texte) Ctrl+C
Coller (un texte) Ctrl+V
Couper (un texte) Ctrl+X
Tout sélectionner (le texte) Ctrl+A
Annuler la dernière action Ctrl+Z
Rétablir la dernière action Ctrl+Y
Fermer une fenêtre Alt+F4
Renommer un fichier F2
Mettre en gras un texte Ctrl+B
Ouvrir l’explorateur de fichiers de Windows Win (touche Windows) + E
Ouvrir l’application Exécuter Win (touche Windows) + R
Afficher le bureau Win (touche Windows) + D
Agrandir la fenêtre active Win (touche Windows) + ↑ (depuis Windows 7)
Minimiser la fenêtre active Win (touche Windows) + ↓ (depuis Windows 7)
Afficher la fenêtre en plein écran F11 (touche Windows)
Copier dans le presse papier la copie de tout l’écran Impr écran ou Ctrl+Impr écran

Attention, le nom de la touche pour faire une capture d’écran peut varier selon les claviers (« Impr Ecran« , « Impécr syst« , « Imp Ecran » etc…)

Copier dans le presse papier la copie d’écran de la fenêtre active Alt+Impr écran
